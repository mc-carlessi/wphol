Format application server's private key

  1. Go to the OCI ORM job page that created the application server and click on its Job resources link.
  2. In Job resources scroll down to the item named tls_private_key.public_private_key_pair and click on its Copy link.
  3. Paste the copied information into the text area below,
  4. Click on Submit.
  5. A new page will be shown detailing the properly formatted key. Copy all of the text and save it to a file on your pc, e.g. wordpress.pem
